#作业一 
##找茬
>平时会用极客学院看一些教学视频,所以查看的他的html原代码。
我认为，在源代码中
```html
<div id="wrapper">
	<div id="header"></div>
	<div id="pager"></div>
	<div id="footer"></div>	
</div>

```
>这种大体结构和标签的使用不够语义化，因该使用HTML5的标签使其更语义化
```html
<div id="wrapper">
	<header></header>
	<article></article>
	<footer></footer>
</div>
```
并且，在img标签中，均没有加入alt对图片加以说明,不符合无障碍性。
极客学院大部分的结构还是很符合语义化的；比如：内容部分的每一小块使用的是section标签包裹列表，很清楚的表达了内容。
